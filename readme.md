# Weather App

## 簡介
這是一個簡單的天氣應用程式，使用 JavaScript 和 Fetch API 從 OpenWeatherMap API 獲取實時天氣數據並顯示在頁面上。用戶可以輸入城市名稱來查詢天氣資訊，並自動保存用戶的喜愛城市。

## 功能
- 輸入城市名稱並顯示天氣資訊
- 顯示溫度、天氣狀況、濕度和風速
- 使用 Local Storage 自動保存和加載用戶的喜愛城市
